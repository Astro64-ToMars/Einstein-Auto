# Einstein-Auto: Relativistic GNC Autonomy Framework
**Research Intent:** Utilizing probabilistic-state evolutionary control (QIGA) to maintain trajectory stability in non-linear force fields.

## Project Structure
- **/src**: Core C/C++ Logic & RK4 Integrator
- **/api**: Flask-based Scenario Swapping Interface
- **/docs**: Annotated Bibliography, Lab Notebook, and Mathematical Derivations
- **/hardware**: 3D models and ESP32 Firmware for Laser-in-the-Loop Validation

## Technical Stack
- **Physics**: 4th-order Runge-Kutta (RK4) Integration of Schwarzschild Geodesics
- **AI**: Quantum-Inspired Genetic Algorithm (QIGA)
- **Acceleration**: Intel oneAPI / OpenCL Parallel Kernels
